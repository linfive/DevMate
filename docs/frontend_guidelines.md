# Frontend Development Guidelines for AI Assistants

## 1. Tech Stack
- Framework: React with Vite.
- Styling: Tailwind CSS.
- Icons: Lucide React.

## 2. Component Design
- Use functional components with Hooks.
- Prefer Tailwind for all styling needs.
- Ensure responsive design using Tailwind's breakpoints (sm, md, lg, xl).

## 3. Best Practices
- Keep components small and focused.
- Use meaningful variable and function names.
- Ensure accessibility (ARIA labels, keyboard navigation).

## 4. Map Integration
- When displaying maps (e.g., for hiking trails), use Leaflet or Mapbox.
- Ensure markers are clearly visible and interactive.
